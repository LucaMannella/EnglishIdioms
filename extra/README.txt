Ho realizzato questo progetto utilizzando 
Android Studio 2.0
con Java JDK 1.8 e gradle 2.10

Istruzioni per importare l'app in Android Studio:
1) Estrarre l'archivio EnglishIdioms.zip in una cartella a piacere
2) Aprire Android Studio
3) Cliccare sul menù File -> New -> Import Project
4) Navigare fino alla cartella in cui è stato estratto l'archivio
5) Cliccare sulla cartella contente il progetto
6) Attendere che il progetto venga importato in Android Studio


Istruzioni per eseguire l'applicazione (avendo un device Android):
1) Cliccare sul menù Run -> Run 'app'
2) Selezionare il dispositivo dal menù che verrà fuori
3) Eventualmente cliccare su "Use same emulator for future launches" se si prevedono più lanci
4) Attendere che l'applicazione venga installata sul dispositivo ed eseguita

P.S.: in caso il device non venga automaticamente riconosciuto:
	1) Recarsi sul sito web del produttore del dispositivo
	2) Scaricare il driver relativo al dispositivo
	3) Installare il driver
	
Istruzioni per eseguire l'applicazione (usando un Emulatore):
1) Cliccare sul menù Run -> Run 'app'
2) Selezionare un emulatore dalla lista
3) Eventualmente cliccare su "Use same emulator for future launches" se si prevedono più lanci
4) Attendere che l'applicazione venga installata sul dispositivo ed eseguita

P.S.: in caso non si disponga di un emulatore
	1) Cliccare su "Create new emulator"
	2) Scegliere un dispositivo dalla lista e cliccare su "Next"
	3) Selezionare una versione delle API (il progetto è compatibile da API 15 in su) e cliccare ancora su "Next"
	4) Cliccare su "Finish"
